    Templating

    All the pages so far have been boring plaintext. Where's all the HTML? Flask "templating" lets you generate and serve HTML with dynamic content

    Setup

    Create a directory called templates/ next to your server.py. Inside it, create a file called home.html.

    mkdir templates
touch home.html

    Make your template look like this.

    from flask import Flask, request, render_template

app = Flask(__name__)
  • ¶

    Rendering a template

    Instead of just returning a string, we have to "render" the template that we made. Our template accepts a variable called name, which we can pass to the render function.

    @app.route("/")
def homepage():
    name = request.args.get('name')
    return render_template('home.html', name=name)
