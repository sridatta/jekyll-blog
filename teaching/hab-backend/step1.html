    Routing

    The server receives a request and checks the URL. Responds differently based on the URL requested:

    • serves requested page
    • responds with an error for invalid URL
    • asks to user to login
    from flask import Flask

app = Flask(__name__)
  • ¶

    Your first route

    We can create a route in Flask by creating an annotation before a function. The annotation tells Flask to call this function when the URL is requested.

    Here we're saying "Call homepage() when someone asks for / "

    @app.route("/")
def homepage():
    return "Hello World!"
  • ¶

    We can create multiple routes and map each one to a different method. Let's make it so user's who go to /create get a different page

    @app.route('/create')

    return "We'll fill this in later"
