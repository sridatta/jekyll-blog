    Dynamic routing

    from flask import Flask

app = Flask(__name__)

@app.route("/")
def homepage():
    return "Hello World!"

@app.route('/create')
def create():
    return "We'll fill this in later"

    Hardcoding every possible URL would get tedious.

    You can tell Flask to respond to a certain URL pattern. The variable parts of the URL get passed as arguments to your function.

    @app.route('/posts/<post_id>')
def show_post(post_id):
    return "You asked for post %s" % (post_id)
