    Query Strings

    You often want to let the client pass some data to the server (search queries, etc) You can accomplish that by using query strings. You've probably seen them before.

    from flask import Flask, request
app = Flask(__name__)
  • ¶

    Accessing Query Strings in Flask

    Flask gives you a request object. request.args contains the query. You can get the data and do whatever you want with it.

    @app.route("/")
def homepage():
    name = request.args.get('name')
    return "Hello %s!" % (name)

    Passing Query Strings

    In your browser, go to http://localhost:5000/?name=sridatta

    The ? says "The query string starts here!"

    name=sridatta is a key-value pair

    You can have multiple key-value pairs seperated by an &. E.g: ?name=sridatta&age=22

    @app.route('/create')
def create():
    return "We'll fill this in later"

@app.route('/posts/<post_id>')
def show_post(post_id):
    return "You asked for post %d" % (post_id)
